Como posso resolver este problema?

Este problema é uma mistura de dois outros desafios que você já encontrou aqui: o 'Maior elemento da lista' e o 'Soma item de uma lista'.

Abra os em outra aba e procure partes de código que podem te ajudar.

Mas como vou saber o tamanho de cada palavra?

Para isso, existe um comando que chama len(), que serve para retornar o tamanho de uma lista ou string.

Exemplo: len('gato')=4 ou len([1,2,4]) = 3
