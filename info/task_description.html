Enunciado

Ana, uma escritora, está revisando um texto e precisa encontrar a palavra mais longa em uma lista de palavras para usá-la como título de um novo capítulo. Se houver mais de uma palavra com o mesmo comprimento, ela escolherá a que aparecer primeiro na lista.

Crie uma função que receba uma lista de palavras como argumento e retorne a maior palavra dessa lista. Se tiver mais de uma palavra com a mesma quantidade de letras, retornar a palavra mais próxima do início da lista.

Entrada: Uma lista de palavras (strings).

Saída: A maior palavra dessa lista.

Exemplo:

{% if interpreter.slug == "js-node" %} 
countConsecutiveSummers(42) == 4
countConsecutiveSummers(99) == 6
{% else %} 
maior_palavra(['amor', 'vida', 'paz']) == 'amor'
maior_palavra(['azul', 'amarelo', 'vermelho', 'verde']) == 'vermelho'
maior_palavra(['gato', 'cachorro', 'passarinho']) == 'passarinho'
{% endif %}
